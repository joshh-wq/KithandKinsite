## Project Context
This is the marketing website for Kith & Kin, a member-owned digital cooperative social app.

- **Framework**: Astro
- **Styling**: Vanilla CSS with CSS variables (no Tailwind)
- **Deployment**: Vercel (auto-deploys on push to main)
- **Backend**: Supabase for waitlist signups

## Key Design Tokens
- Primary accent: `--sage` (#5A6B52)
- Secondary accent: `--brown` (#B3865B)
- Background: `--cream` (#F5F2E9)
- Fonts: Lora (serif headings), Inter (body)
